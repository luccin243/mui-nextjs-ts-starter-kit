# MUI NextJs using TypeScript Starter Kit

## Features

✅ MUI

✅ Theme provider: Dark and Light with auto detect of user preferences

✅ Redux

✅ Localizations with auto detect of user preferences

✅ SEO

✅ Security Headers

✅ Base URL and Paths : @/fields/\*

✅ Date parser

## Getting Started

```bash
npm install

npm run dev
```
